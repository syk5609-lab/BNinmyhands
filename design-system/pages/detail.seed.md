# Detail Page Override Seed

## Page goal
Explain one symbol in the same-run context so the user understands why it surfaced and what changed.

## Composition
Suggested order:
1. symbol header + run context
2. score overview
3. latest + delta context
4. why this coin
5. funding context
6. recent history
7. discussion
8. sponsor slot

## Priority
The page should read like a coherent analysis story.
Not a stack of unrelated widgets.

## Funding block rules
Funding is explanatory context.
Show:
- latest funding
- absolute funding
- bias
- interpretation

These should read as one interpretive block.

## Discussion rules
- compact
- clearly below the analysis
- useful for interpretation support only
- report/delete affordances must be unobtrusive but obvious

## Sponsor rules
- detail bottom only
- clearly Sponsored
- never styled like a score/funding/history block
